- **프로젝트 이름** : 동구라미

### 🟡 주요 기능 

**동아리 지원자**
   - 동아리 지원
   - 동아리 지원 결과 푸시 알림

**동아리 회장**
   - 동아리 관리
   - 동아리 지원자 합격 관리 - 합격/불합격

**중앙 동아리 연합회**
   - 중앙 동아리 관리 - 추가, 삭제
